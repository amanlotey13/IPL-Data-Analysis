# IPL Data Analysis Dashboard

## Project Overview
Excel-based dashboard analyzing IPL team and player performance across seasons.

## Tools Used
- Microsoft Excel

## Key Objectives
- Analyze team performance
- Compare player statistics
- Study season-wise trends

## Key Insights
- Winning patterns identified
- Top-performing teams analyzed
- Player statistics visualized
